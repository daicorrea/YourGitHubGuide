Starting With Git

About GitHub:

Github is a Version Control Repository. It's a web-based Git and it offers Source Code Management, Wikis, Bug Tracking and Feature Requests.

The site also provides a social network vibe that allows the teams to use feeds, followers and graphs to analyse the team work on the project.

To use it on your computer you need to download a version. You can choose between the Desktop version or the simple Git Command Line.

Creating an Account:

To create a new external account, you have to go to the website, fill the form and Signup for Github.

When you have a company Github, you need to go to your company's Github Account and enter your intranet information.

Creating a New Organization:

If you want to have a project with a team, you can create an organization and add whoever you want.

If you are in an external account, you have to choose your organization plan and enter your billing information.

If you are in a company account, the organization will be in your company's domain and it will only request the organization name and a contact email.

Configuring the Environment:

To start using Github on your machine, you need to have a SSH key connecting your account with your computer. For this you need to follow some steps:

  • Create an SSH Key on your computer;
  • Connect the SSH key with your account on Github;
  • Test your SSH connection.
How you do each of these steps you can check here.
